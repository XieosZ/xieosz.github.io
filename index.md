To download News-App, follow these steps:

1. **Visit the Releases Page**  
   Click this link: [Download News-App](https://github.com/XieosZ/News-App/releases). This link will take you to the page where you can find the latest version.

2. **Choose the Latest Version**  
   On the Releases page, look for the section titled "Latest Release." This section will display the newest version of the app.

3. **Download the App**  
   Under the latest release, you will see several files. Look for the one that matches your operating system:
   - If you are using Windows, you might find a file named something like `News-App-windows.exe`.
   - If you are using macOS, look for `News-App-mac.dmg`.

   Click on the appropriate file to start the download.

4. **Install the App**  
   Once the download is complete, follow these steps to install:
   - For Windows:
     1. Locate the downloaded file (usually in your Downloads folder).
     2. Double-click on the `News-App-windows.exe` file.
     3. Follow the on-screen instructions to install the app.
   - For macOS:
     1. Locate the downloaded file (`News-App-mac.dmg`).

     3. Drag the News-App icon to your Applications folder.

5. **Launch News-App**  
   After installation, you can find News-App in your Start menu (on Windows) or Applications folder (on macOS). Click on the app icon to launch it.

6. **Start Reading**  
   Now that News-App is running, you can choose which news categories you want to read from. Explore various articles and stay updated!
